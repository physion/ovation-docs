Create a Project¶

Once you’ve installed Ovation, you’re ready to start organizing your data. Ovation organizes data in a fashion familiar to scientists including projects, experiments, and measurements. The best way to get started is to create a project.

Log in¶

Before you can add data to your Ovation database, you must login using your email address and ovation.io password. If you've forgotten your ovation.io password, you can reset it.

Create a Project¶

Projects are top-level entities in the Ovation data model, and are used to organize related Experiments. Projects may contain many Experiments, and Experiments can belong to more than one Project.
