******
Search
******

Ovation's powerful and intuitive search engine helps you find items in the database by keyword, property or protocol parameter. Search queries are specified using a simple text syntax. Besides the the elements described below, Ovation's query syntax supports full boolean logic so you can use ``&&`` (AND) ``||`` (OR) and ``!`` (NOT) as well as ``(`` and ``)`` to group statements.
